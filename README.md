karma example
=====================================================================
Example usage of Karma test running utilising Jasmine, Sinon and Babel

Installation
---------------------------------------------------------------------
### Prerequisites
- Node v4.4.7+
- NPM 4.1.1+

Install all dependencies
```
npm install
```

Usage
---------------------------------------------------------------------
- Spec files are create in the ./tests folder
- Code files are created in the ./js folder

Run Karma
```
karma start
```
